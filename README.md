## Lua GG Encryption

Encryption tool for Lua Game Guardian scripts.

![Menu](https://github.com/islavikfx/LuaGG-Encryption/blob/main/img/picture.png?raw=true)

Encryption tool includes good obfuscation formula and protection against AntiLoad, AntiLog, AntiHook, AntiSSTools and AntiLasm.
